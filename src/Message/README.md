A simple component to display a message

### Usage

```jsx
<>
  <Message type="info" body="This is an informative message" title="Info" />
  <Message type="warning" body="This is a warning message" title="Warning" />
  <Message type="error" body="This is an error message" title="Error" />
  <Message type="info" body="This is an informative message (without header)" />
  <Message type="warning" body="This is a warning message (without header)" />
  <Message type="error" body="This is an error message (without header)" />
</>
```
