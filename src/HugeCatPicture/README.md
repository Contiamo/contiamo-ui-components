# Basic Usage

Use this component like so

```jsx
<div style={{ display: "flex" }}>
  <HugeCatPicture size={400} borderColor="#f00" />
  <HugeCatPicture size={400} borderColor="primary" />
  <HugeCatPicture size={400} />
  </div>
<div style={{ position: "absolute", top: 175, left: 150, display: "flex" }}>

  <HugeCatPicture size={400} borderColor="color.background.lightest" />
  <HugeCatPicture size={400} spin />
</div>
```
